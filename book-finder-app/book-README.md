# Book Finder App

A responsive web application that searches
for books using Open Library API.

## Features

- Search for books
- Loading spinner
- Error handling
- Responsive design
- Dynamic DOM rendering

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API

## API Used

https://openlibrary.org/search.json

## How to Run

1. Download project
2. Open with VS Code
3. Run using Live Server

A sample database schema is provided in the `book-schema.sql` file.

- Store searched book titles
- Store search queries
- Save search date and time
